**PROCESS:**

1.  Read the user's data from `{markdown_resume}`.
2.  Read the `{latex_template}`.
3.  Go through the template section by section, paying close attention to the two-column layout.
4.  For each section, REMOVE the template's example text and INSERT the corresponding user data.

    *   `\namesection{John}{Doe}{...}`
    *   `\runsubsection{Company Name}`
    *   `\descript{| Job Title}`
    *   `\location{Date Range | Location}`
    *   `\begin{tightemize}` for bullet points.

**INPUTS:**

**1. User Resume Data (Markdown):**
```markdown
{markdown_resume}
```

**2. LaTeX Template File (Deedy CV):**
```latex
{latex_template}
```
