**LaTeX Formatting Guidelines:**

1. **Header Section**:
```latex
\begin{center}
    \textbf{\Huge \scshape FULL NAME} \\ \vspace{1pt}
    \small \href{tel:PHONE}{\textcolor{blue}{\underline{PHONE}}} $|$ \href{mailto:EMAIL}{\textcolor{blue}{\underline{EMAIL}}} $|$ 
    \href{LINKEDIN_URL}{\textcolor{blue}{\underline{LinkedIn}}} $|$ 
    \href{GITHUB_URL}{\textcolor{blue}{\underline{GitHub}}}
\end{center}
```

2. **Summary Section**:
```latex
\section{Summary}
\small{Enhanced summary text here with \textbf{bold keywords} for emphasis}
```

3. **Experience Section**:
```latex
\section{Experience}
\resumeSubHeadingListStart
    \resumeSubheading
      {Job Title}{Date Range}
      {Company Name}{Location}
      \resumeItemListStart
        \resumeItem{Achievement or responsibility with \textbf{technologies} highlighted}
        \resumeItem{Quantifiable achievement showing impact}
      \resumeItemListEnd
\resumeSubHeadingListEnd
```

4. **Projects Section**:
```latex
\section{Projects}
\resumeSubHeadingListStart
    \resumeProjectHeading
        {\textbf{Project Name} $|$ \emph{Context/Type}}{}
        \resumeItemListStart
            \resumeItem{Detailed implementation point with \textbf{key technologies}}
            \resumeItem{Technical challenge solved and approach taken}
            \resumeItem{Tools: \textit{\textbf{Tech1}, \textbf{Tech2}, Tech3}}
        \resumeItemListEnd
\resumeSubHeadingListEnd
```

5. **Education Section**:
```latex
\section{Education}
\resumeSubHeadingListStart
    \resumeSubheading
      {University Name}{Location}
      {Degree}{Date Range}
\resumeSubHeadingListEnd
```

6. **Skills Section** (Dynamic Categories):
```latex
\section{Technical Skills}
\begin{itemize}[leftmargin=0.15in, label={}]
    \small{\item{
     \textbf{Category Name}{: Skill1, Skill2, Skill3, Skill4} \\
     \textbf{Another Category}{: Skill5, Skill6, Skill7} \\
    }}
\end{itemize}
```

**Special Formatting Rules:**
- Use \textbf{} for technologies and important keywords
- Use \textit{} for context or secondary information
- Use \emph{} for subtle emphasis
- Escape special characters: & → \&, % → \%, $ → \$, # → \#, _ → \_
- For URLs, always use \href{URL}{\textcolor{blue}{\underline{Display Text}}}
- Maintain consistent spacing with \vspace{-2pt} or \vspace{-5pt} as needed
- Use $|$ for pipe separators in headers

**Content Guidelines:**
- Keep descriptions concise and impactful
- Start bullet points with strong action verbs
- Include quantifiable metrics where provided
- Highlight technologies naturally within sentences
- Ensure all content fits well within page margins
