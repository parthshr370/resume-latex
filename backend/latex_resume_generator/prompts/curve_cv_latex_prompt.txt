2.  Read the `{latex_template}`.
3.  Go through the template section by section (Header, Profile, Skills, Experience, etc.).

    *   In `\leftheader{}`, use `\firstname{}`, `\lastname{}`, `\email{}`, etc.
    *   Use `\begin{eventlist}` for Experience and Education.

```markdown

```

```latex
{latex_template}
```
